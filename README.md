# Axonscut 🎬
**高效、隱私、免上傳的線上影音剪輯工具**

Axonscut 是一個專為快速處理影音需求設計的網站，核心理念是「本地處理」。使用者無需登入、無需將檔案上傳至雲端伺服器，即可直接在瀏覽器進行影音剪輯。

## 🎯 專案目標
* **隱私至上**：檔案不經過伺服器，完全在使用者本地瀏覽器完成運算。
* **即時便利**：打破傳統剪輯軟體需安裝、需登入的門檻，開啟網頁即刻剪輯。

## 🛠 技術挑戰與解決方案 (Troubleshooting)

### 1. COOP/COEP 跨來源隔離限制
* **問題原因**：
    為了防範 Spectre 等資安攻擊，瀏覽器會限制未開啟「跨來源隔離狀態」的環境使用 **共享記憶體** 或 **高精度時間測量** 的功能，這導致 FFmpeg.wasm 等剪輯核心組件無法運作。
* **技術瓶頸**：
    GitHub Pages 為靜態託管，不支援自訂 HTTP Response Headers，無法手動開啟隔離政策。
* **解決方案**：
